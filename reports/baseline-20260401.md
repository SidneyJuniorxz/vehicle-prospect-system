# Baseline de Coleta — 2026-04-01

Objetivo: medir rapidamente a completude de preço e contato nos anúncios já coletados na base `vehicle_prospect` (PostgreSQL) antes de ajustar scrapers e fluxo de coleta.

Consulta usada
```sql
select
  source,
  count(*) as total,
  sum(case when price is not null then 1 else 0 end) as with_price,
  sum(case when contact_info is not null and contact_info<>'' then 1 else 0 end) as with_contact
from vehicle_ads
group by source
order by source;
```

Principais gaps observados
